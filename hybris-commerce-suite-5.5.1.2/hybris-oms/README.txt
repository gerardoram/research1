Contents of Release
---------------------

This section contains a general summary of the files and directories in the hybris Order Management System (OMS) Version 5.5.1.0 release.

 --> binary:
  contains pre-packaged web applications (WAR) and RESTful client libraries.
  Use those if you do not plan to extend OMS
  
 --> dependencies:
  contains all dependencies needed to compile and install the software in this release
 
 --> documentation:
  contains generated API documentation
  
 --> sdk:
  contains the archetype to be used as standard extension point
 
 --> tools:
  contains additional applications to support you OMS installation
  - dataonboarding: application to load data into an OMS installation
  - dataonbaording rest client: pre-compiled client to remote-control your data onboarding application
