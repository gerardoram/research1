You can import data to OMS using the dataonboarding service. The dataonbarding service is located under release\tools\dataonboarding\binary\webapp
To import data:
1) Deploy and configure the OMS distribution, see https://wiki.hybris.com/display/release5/Simplified+Installation+of+Order+Management+Services+with+Data+Onboarding
2) Deploy the dataonboarding web application to Tomcat
3) To import data, please follow the instructions at https://wiki.hybris.com/display/release5/Installing+Order+Management+Services+with+Data+Onboarding
