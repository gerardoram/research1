About the configurations in this folder
=========================================

Please modify/copy all 4 files in to your Tomcat "lib" directory. This is the default directory used by the core+ web applications.

List of files to copy
======================
com.hybris.oms_oms-rest-webapp.properties						--> OMS Service configuration
com.hybris.oms_oms-rest-webapp-logback.xml						--> OMS logging configuration
com.hybris.dataonboarding_dataonboarding-webapp.properties		--> Data Onboarding Service configuration
com.hybris.dataonboarding_dataonboarding-webapp-logback.xml		--> Data Onboarding logging configuration


Notes
======
To activate good logging, please adjust the appender to write to a logfile in your location of choice
